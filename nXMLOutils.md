### χrust (markup-rs)

> A Rust implementation of the [XQuery and XPath Data Model 3.1](https://www.w3.org/TR/xpath-datamodel-31/) and [XSLT 3.0](http://www.w3.org/TR/xslt-30/). The idea is to separate the syntax from the semantics. A [Transform](https://docs.rs/xrust/latest/xrust/transform/enum.Transform.html) performs the semantics; an XPath expression or XSL Stylesheet is the syntax that is mapped to a [Transform](https://docs.rs/xrust/latest/xrust/transform/enum.Transform.html).

Cf. https://gitlab.gnome.org/World/Rust/markup-rs

https://docs.rs/xrust/latest/xrust/

https://gitlab.gnome.org/World/Rust/markup-rs/xrust

## GO

https://doc.speedata.de/goxml/

Three interrelated Go packages for XML processing:

- **[goxml](https://github.com/speedata/goxml)** — XML parser and tree model
- **[goxpath](https://github.com/speedata/goxpath)** — XPath 2.0+ evaluator with XPath 3.1 functions
- **[goxslt](https://github.com/speedata/goxslt)** — XSLT 3.0 processor
